HJ Landscaping Website Setup

1. Upload all files (index.html, style.css, script.js) to your GitHub repository:
   https://github.com/YOUR_USERNAME/hjlandscaping.github.io

2. Replace:
   - YOUR_EMAIL in index.html & script.js → with your real email (FormSubmit)
   - YOUR_CALENDAR_LINK in index.html → with your Calendly booking link
   - YOUR_PHONE in index.html → with your phone number for Call Now button

3. Go to repository Settings → Pages → ensure Branch is set to 'main'

4. Your website will be live at:
   https://hjlandscaping.github.io

5. To connect your domain:
   - Create CNAME file → write: hjlandscaping.com.au
   - Add these A records in your domain DNS:
     185.199.108.153
     185.199.109.153
     185.199.110.153
     185.199.111.153
